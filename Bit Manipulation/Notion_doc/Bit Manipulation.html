Property-1:

X = number of set bits in A
Y = number of set bits in B
Z = number of set bits in A^B
Then,
Z is even if A + B is even. Else, odd - odd.

Property-2:

A + B = ( A ^ B ) + 2 ( A & B )
A + B = ( A | B ) + ( A & B )

Property-3:

The formula x &− x sets all the one bits to zero, except for the last one bit.
The formula x | ( x − 1) inverts all the bits after the last one bit.

Given N integers, where every integer appears twice and two integers appear once. Extract them.

C++
Copy
Caption
// Use a hashmap, T.C - O(nlogn) S.C - O(n) // My soln. Intuition - First get the xor a ^ b. Now, there'll be atleast one set bit in this value. Based on that divide the array into two pieces. How to get that bit? In my case, the set I'm going to consider is the last set bit. So, two obtain that: xor = a ^ b; xor_copy = xor; xor = xor & ( xor - 1 ) // This removes the last set bit in xor. xor = xor ^ xor_copy // This is the last set bit. class Solution { public: vector<int> singleNumber(vector<int>& arr) { long long int xor_val = 0; int n = arr.size(); for( int i = 0 ; i < n ; i ++) xor_val = xor_val ^ arr[i]; long long int xor_copy = xor_val; xor_val = xor_val & ( xor_val - 1); xor_val = xor_val ^ xor_copy; int a = 0 , b = 0; for( int i = 0 ; i < n ; i ++){ if( arr[i] & xor_val ) a = a ^ arr[i]; else b = b ^ arr[i]; } vector<int> ans = { a , b }; return ans; } };
